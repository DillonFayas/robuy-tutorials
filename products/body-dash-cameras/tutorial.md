## Installing your Body/Dash Cameras

Download the asset. If you purchased through Robuy, it was sent to your DMs on Discord. If you purchased with Payhip, you can retrieve it from your email.

Open a **blank** baseplate project in Roblox Studio and drop the file in.

## Team Whitelist

First, install Poncake's Plugin, linked below. Then, you can use the plugin to edit the team whitelist script found in `Workspace > Computer and edit the TeamWhitelist`. In the plugin, navigate to the Team Editor module, select the `TeamWhitelist` ModuleScript, then click "Use Current Selection" to begin editing. Add as many teams as you want to grant access to the cameras.

## Body Cameras:
Take either the folder or the model named “BodyCamera” and place it into a vest giver, or whatever method you use to distribute the cameras amongst users. Note that you may want to disable the script named “SetupCamera” until the camera has been given to the user.

## Dash Cameras:
Either the folder or the model named “DashCamera” can be placed into a car model. A-Chassis will not allow folders which is why the models are provided. If you are using a car spawner, then assuming that the car is being stored in ServerStorage, there should be no need to disable the “SetupCamera” script.
